## 🎯 Реализованная функциональность

### ✅ 1. Редактирование всех ячеек
**Что изменилось:**
- Раньше: только некоторые ячейки (числовые данные) были редактируемыми
- Теперь: **ВСЕ ячейки** можно редактировать, включая наименования, номера строк, формулы

**Как работает:**
- Каждая ячейка содержит `<input type="text">`
- При фокусе ячейка подсвечивается синим
- Изменения автоматически сохраняются в localStorage

### ✅ 2. Добавление новых строк
**Что добавлено:**
- Кнопка **"➕ Добавить новую строку"** под каждой таблицей
- При нажатии создается новая пустая строка
- Автоматическая генерация уникальных ID для ячеек
- Фокус переходит на первую ячейку новой строки

**Особенности:**
- Определяет количество столбцов автоматически
- Сохраняет структуру таблицы
- Данные сразу готовы к вводу

### ✅ 3. Удаление строк
**Что добавлено:**
- Кнопка **"🗑️"** в конце каждой строки (кроме заголовка)
- Подтверждение перед удалением
- Автоматическая очистка данных из localStorage
- Пересчет данных после удаления

## 🚀 Как использовать

### Для пользователей:

1. **Редактировать ячейку:**
   - Кликните в любую ячейку
   - Введите/измените значение
   - Данные сохранятся автоматически

2. **Добавить строку:**
   - Нажмите "➕ Добавить новую строку" под таблицей
   - Заполните новую строку данными

3. **Удалить строку:**
   - Нажмите 🗑️ в конце строки
   - Подтвердите удаление

### Для разработчиков:

#### Быстрая интеграция в любой HTML файл:

```html
<!-- 1. Подключить модуль -->
<script src="editable-table.js"></script>

<!-- 2. Инициализировать после загрузки -->
<script>
document.addEventListener('DOMContentLoaded', () => {
    // Ваша логика загрузки данных
    loadData();
    
    // Инициализация редактируемых таблиц
    setTimeout(() => {
        initAllEditableTables('table', handleInput);
    }, 100);
});

function handleInput(cellId) {
    // Ваша логика при изменении
    calculateTotals();
    saveData();
}
</script>
```

## 🔧 Технические детали

### Хранение данных
- **Формат:** localStorage с ключами вида `tableId_rowIndex_colIndex`
- **Пример:** `sec1_row5_col2` = "1000"
- **Автосохранение:** После каждого изменения через debounce

### Генерация ID
```javascript
// Для секционных таблиц (rb-svodnaya.html)
`sec${sectionNum}_row${rowIndex}_col${colIndex}`

// Для обычных таблиц
`${tableId}_row${rowIndex}_col${colIndex}`
```

### Структура строки
```html
<tr id="table_row_5">
  <td><input class="editable-cell" id="table_row5_col0" value="1"/></td>
  <td><input class="editable-cell" id="table_row5_col1" value="Расход"/></td>
  <td><input class="editable-cell" id="table_row5_col2" value="1000"/></td>
  <td><button class="btn-delete-row" onclick="deleteTableRow(...)">🗑️</button></td>
</tr>
```
