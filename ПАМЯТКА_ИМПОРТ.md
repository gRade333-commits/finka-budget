# ✅ Быстрая проверка системы импорта

## 🚀 Порядок действий

### 1. Импорт данных
- Откройте: [import-excel.html](import-excel.html)
- Загрузите файлы: "Бюджет РБ 2026 (1).xlsx" и "Бюджет ПУ 2026 (1).xlsx"

### 3. Проверка формул
- Откройте: [check-formulas.html](check-formulas.html)
- Загрузите любой Excel файл
- Нажмите: **"Анализировать формулы"**
- Просмотрите все формулы

### 4. Проверка на странице
- Откройте: [rb-svod.html](rb-svod.html)
- Выберите филиал: Алматы
- Проверьте: данные отображаются

### 5. 🆕 Управление формулами после импорта
- **После импорта**: формулы консолидации автоматически отключены

- **Активация**: начните редактировать любую ячейку
- **Подробности**: см. [ПАМЯТКА_ИМПОРТ_ФОРМУЛЫ.md](ПАМЯТКА_ИМПОРТ_ФОРМУЛЫ.md)

## ⚠️ Частые проблемы

| Проблема | Решение |
|----------|---------|
| Данные не импортируются | Очистите localStorage, повторите импорт |
| Неверные данные на rb-svodnaya | Проверьте наличие svodnaya_sections_full.json |
| Лист не найден | Проверьте название листа в Excel (могут быть пробелы) |
| Ошибки в консоли | Откройте F12, проверьте консоль браузера |
