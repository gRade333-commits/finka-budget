🔄 Миграция данных localStorage

ℹ️ Информация

Данные сохранялись со старыми ключами, а консолидация ищет новые. Эта утилита перенесёт данные на правильные ключи.

Старые данные будут удалены после успешного переноса.

Будут выполнены следующие миграции:

svod_rb_* → rb_svod_*
income_rb_* → rb_income_*
svod_fzp_* → rb_fzp_*
kalkulyacia_2026_* → rb_kalkulyacia_*
plan_komandir_* → rb_plan_komandir_*
svodnaya_rb_* → rb_svodnaya_*

* - означает almaty, uralsk, astana
