## 🎯 Процесс импорта

1. **Загрузка файлов**: Пользователь выбирает 2 Excel файла
2. **Парсинг**: SheetJS читает все листы
3. **Проверка**: Система находит нужные листы по именам
4. **Конвертация**: Данные преобразуются в формат страниц
5. **Сохранение**: Данные записываются в localStorage
6. **Проверка**: Пользователь проверяет импорт

## ⚠️ Важные моменты

### Формулы из Excel

- ✅ Формулы **сохраняются** в библиотеке SheetJS
- ✅ Значения формул импортируются
- ⚠️ Формулы не выполняются в браузере (импортируются вычисленные значения)

### Форматы ячеек

- **Числа**: Автоматическая конвертация в Number
- **Текст**: Сохраняется как String
- **Пустые**: Конвертируются в 0 или '' в зависимости от контекста
- **Формулы**: Импортируются вычисленные значения

### Производительность

- Импорт 17 листов занимает ~3-5 секунд
- rb-svodnaya (641 строка) требует дополнительной обработки
- Используется async/await для асинхронной загрузки

## 🐛 Устранение проблем

### Проблема: Данные не импортируются

**Диагностика**:
1. Проверьте консоль браузера (F12)
2. Убедитесь что файлы Excel не повреждены
3. Проверьте названия листов в Excel

**Решение**:
1. Очистите localStorage через [clear-all-data.html](clear-all-data.html)
2. Повторите импорт
3. Проверьте логи на странице импорта

### Проблема: rb-svodnaya показывает неверные данные

**Диагностика**:
1. Проверьте наличие файла svodnaya_sections_full.json
2. Проверьте формат данных в localStorage (ключ `rb_svodnaya_almaty`)
3. Убедитесь что используется правильный формат ID: `sec#_row#_col#`

**Решение**:
1. Убедитесь что svodnaya_sections_full.json доступен
2. Повторите импорт
3. Проверьте образец данных в [check-import.html](check-import.html)

### Проблема: Формулы не работают

**Пояснение**:
- Формулы из Excel **не выполняются** в браузере
- Импортируются **вычисленные значения** формул
- Для пересчёта формул нужно использовать JavaScript

**Решение**:
- Это нормальное поведение
- Система импортирует результаты вычислений из Excel
- Для динамических расчётов используйте формулы на страницах
