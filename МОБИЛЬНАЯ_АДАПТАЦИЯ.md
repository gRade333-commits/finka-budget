## 🎨 Рекомендации по использованию

### 1. Для больших таблиц
Оберните таблицу в контейнер с классом `.table-container`:
```html
<div class="table-container">
    <table>
        <!-- содержимое таблицы -->
    </table>
</div>
```

### 2. Для адаптивных кнопок
Используйте класс `.btn-group` для группы кнопок:
```html
<div class="btn-group">
    <button class="btn">Кнопка 1</button>
    <button class="btn">Кнопка 2</button>
</div>
```

### 3. Скрытие элементов
```html
<div class="desktop-only">Только на десктопе</div>
<div class="mobile-only">Только на мобильных</div>
```
