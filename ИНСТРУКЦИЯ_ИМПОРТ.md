# 📋 Инструкция по импорту данных из Excel

## 🎯 Для бухгалтера и финансиста

### Шаг 1: Подготовка Excel файлов

Убедитесь, что у вас есть два файла:
1. **Бюджет РБ 2026 (1).xlsx** - Республиканский бюджет
2. **Бюджет ПУ 2026 (1).xlsx** - Подушевое финансирование

### Шаг 2: Открытие страницы импорта

1. Откройте главную страницу системы ([index.html](index.html))
2. В разделе "🛠️ Утилиты и инструменты" нажмите на ссылку **"ИМПОРТ ИЗ EXCEL"**
3. Откроется страница импорта ([import-excel.html](import-excel.html))

### Шаг 3: Загрузка файлов

1. **Республиканский бюджет (РБ)**:
   - Нажмите кнопку "Выбрать файл РБ"
   - Выберите файл "Бюджет РБ 2026 (1).xlsx"
   - Дождитесь сообщения "✓ Файл РБ загружен"

2. **Подушевое финансирование (ПУ)**:
   - Нажмите кнопку "Выбрать файл ПУ"
   - Выберите файл "Бюджет ПУ 2026 (1).xlsx"
   - Дождитесь сообщения "✓ Файл ПУ загружен"

### Шаг 4: Проверка листов (опционально)

Нажмите кнопку **"Показать листы"** чтобы убедиться, что все листы в файлах найдены:

**РБ должен содержать 6 листов:**
- Свод
- Сводная общая 2026г.
- ДОХОДНАЯ ЧАСТЬ РБ
- СВОД ФЗП
- Калькуляция 2026
- План командир 2026

**ПУ должен содержать 11 листов:**
- СВОД общий 2026
- СС Дотация
- ДОХОДНАЯ ЧАСТЬ ПУ
- ДОХОДНАЯ ЧАСТЬ ДТ
- ДОХОДНАЯ ЧАСТЬ доп
- Свод ФОТ Алматы
- График род. оплаты Алматы
- Калькуляция Алматы
- План команд в пред РК
- План команд за пределы РК
- СС Алматы

### Шаг 5: Импорт данных

1. Нажмите кнопку **"🚀 ИМПОРТИРОВАТЬ"**
2. Дождитесь завершения процесса (появится прогресс-бар)
3. Проверьте статистику:
   - **Листов обработано**: должно быть 17 (6 РБ + 11 ПУ)
   - **Создано наборов данных**: должно быть 51 (17 × 3 филиала)
   - **Ошибок**: должно быть 0

### Шаг 6: Проверка импорта

1. Вернитесь на главную страницу
2. В разделе "Утилиты" нажмите **"ПРОВЕРКА ИМПОРТА"**
3. Нажмите кнопку **"🔍 Проверить данные"**
4. Убедитесь, что все данные импортированы:
   - ✅ Все РБ таблицы: найдено 18 (6 × 3 филиала)
   - ✅ Все ПУ таблицы: найдено 33 (11 × 3 филиала)

### Шаг 7: Проверка на страницах

Откройте любую страницу бюджета (например, [rb-svod.html](rb-svod.html)) и убедитесь, что данные отображаются корректно.

## ⚠️ Важные моменты

Данные автоматически импортируются для трёх филиалов:

- **Уральск** (множитель 0.7)
- **Астана** (множитель 0.5)

### Формулы

- ✅ Формулы из Excel **сохраняются**
- ✅ Значения ячеек импортируются корректно
- ✅ Числовые данные конвертируются автоматически

## 🔧 Устранение проблем

### Проблема: "Лист не найден"

**Решение**: Проверьте название листа в Excel файле:
- Убедитесь что название точно совпадает (учитываются пробелы)
- Проверьте что лист не скрыт
- Убедитесь что файл не повреждён

### Проблема: "Данные не отображаются на странице"

**Решение**:
1. Откройте [check-import.html](check-import.html)
2. Проверьте что данные импортированы
3. Проверьте консоль браузера (F12) на ошибки
4. Попробуйте очистить данные и импортировать заново

### Проблема: "Ошибки при импорте"

**Решение**:
1. Убедитесь что файлы не повреждены
2. Проверьте что в Excel нет защиты листов
3. Убедитесь что формат файлов .xlsx
4. Попробуйте пересохранить Excel файлы

## 📞 Поддержка

При возникновении проблем:
1. Проверьте логи на странице импорта
2. Используйте страницу [check-import.html](check-import.html) для диагностики
3. Проверьте консоль браузера (F12 → Console)
