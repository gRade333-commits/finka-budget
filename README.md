## 🔧 Деплой на Vercel

### Вариант 1: Через GitHub (рекомендуется)

1. Создайте репозиторий на GitHub
2. Подключите локальный проект:
```bash
git remote add origin https://github.com/YOUR_USERNAME/finka-budget-system.git
git branch -M main
git push -u origin main
```

3. Импортируйте на Vercel:
   - Зайдите на https://vercel.com
   - Add New Project → Import from GitHub
   - Выберите репозиторий → Deploy

### Вариант 2: Через Vercel CLI
```bash
npm install -g vercel
vercel login
vercel --prod
```
