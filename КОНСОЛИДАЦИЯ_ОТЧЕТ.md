### Для файлов с кнопками филиалов (RB)

```javascript
function switchBranch(branch) {
    saveData();
    currentBranch = branch;
    
    // Обновить активную кнопку
    document.querySelectorAll('.branch-btn').forEach(btn => {
        btn.classList.remove('active');
        if (btn.dataset.branch === branch) {
            btn.classList.add('active');
        }
    });
    
    if (branch === 'consolidated') {
        loadConsolidatedData();
    } else {
        loadData();
    }
}

function loadConsolidatedData() {
    const branches = ['almaty', 'uralsk', 'astana'];
    const allData = {};
    
    // Загрузка данных из localStorage
    branches.forEach(b => {
        const key = `${sheetName}_${b}`;
        const saved = localStorage.getItem(key);
        if (saved) {
            allData[b] = JSON.parse(saved).cells || {};
        }
    });
    
    // Суммирование значений
    document.querySelectorAll('input').forEach(input => {
        const cellId = input.dataset.cell;
        let sum = 0;
        branches.forEach(b => {
            if (allData[b]?.[cellId]) {
                sum += parseFloat(allData[b][cellId]) || 0;
            }
        });
        input.value = sum.toFixed(2);
    });
    
    // Пересчет формул
    calculateTotals();
    document.getElementById('lastUpdate').textContent = 'Консолидация из 3 филиалов';
}
```

## 🐛 Исправленный баг (26.12.2025)

### Проблема
При переключении между городами и консолидацией данные в консолидации постоянно увеличивались. Причина:
1. При переключении филиала вызывался `saveData()` 
2. Если текущий филиал был "consolidated", консолидированные данные сохранялись в localStorage
3. При следующей консолидации эти данные суммировались заново → удвоение/утроение значений

### Решение
**Запретить сохранение данных в режиме консолидации:**

```javascript
// Для RB файлов (с кнопками)
function switchBranch(branch) {
    // НЕ сохраняем данные если текущий режим - консолидация
    if (currentBranch !== 'consolidated') {
        saveData();
    }
    currentBranch = branch;
    // ...
}

// Для PU файлов (с dropdown)
function save() {
    // Не сохраняем в режиме консолидации
    if (branch === 'consolidated') return;
    // ...
}

function saveData() {
    // Не сохраняем в режиме консолидации
    if (currentBranch === 'consolidated') return;
    // ...
}
```
