## Как запустить?

### Вариант 1: Простой HTTP сервер (рекомендуется)

```powershell
python -m http.server 8000
```

Затем откройте в браузере: http://127.0.0.1:8000/rb-svodnaya-full.html

### Вариант 2: Через index-budget.html

Просто откройте главную страницу:
```powershell
Start-Process index-budget.html
```

И выберите "Сводная общая 2026" в разделе РБ.

## Зависимости:

**Важно:** Файл `svodnaya_sections_full.json` должен находиться в той же папке!

- ✅ svodnaya_sections_full.json (6740 строк, 250KB)

## Особенности работы:

1. **Сворачивание разделов:** Кликните на заголовок раздела, чтобы свернуть/развернуть его
2. **Быстрая навигация:** Используйте ссылки вверху страницы для перехода к нужному разделу
3. **Автосохранение:** Данные сохраняются через 1 секунду после ввода
4. **Филиалы:** Переключайтесь между Алматы/Уральск/Астана - у каждого свои данные
5. **Кнопка "Наверх":** В правом нижнем углу для быстрой прокрутки

## Технические характеристики:

- **Размер HTML:** ~18 KB
- **Размер JSON:** ~250 KB (6740 строк)
- **Общий размер разделов:** 572 строки
- **Количество редактируемых полей:** ~3000+
- **Браузеры:** Chrome, Firefox, Edge (последние версии)

## Производительность:

- Загрузка страницы: < 1 сек
- Генерация таблиц: < 0.5 сек
- Автосохранение: debounce 1000ms
- Хранение: localStorage (до 10MB данных)
